# java-lotto-precourse
해당 프로젝트는 간단한 로또 발매기 구현을 목적으로 한다.<br>
이 프로그램은 로또를 구매한 후, 당첨 번호 및 보너스 번호를 입력받아 당첨 통계를 출력하여야 한다.
